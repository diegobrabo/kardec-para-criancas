# 1. Que é Deus?
“Deus é a inteligência suprema, causa primária de todas as coisas”

## Explicação para crianças

Quando o livro diz que Deus é “a inteligência suprema”, ele quer falar de uma mente muito maior do que a nossa, que sabe e organiza tudo de um jeito perfeito. Não é como a inteligência de uma pessoa que aprende aos poucos; é uma inteligência maior que sempre existiu.

E quando diz “causa primária de todas as coisas”, significa “o primeiro motivo”, a primeira origem. É como perguntar: “De onde veio o universo, a vida, as leis da natureza?” A ideia é que Deus seja a origem de tudo isso, antes de qualquer outra coisa existir.

Um exemplo: quando você vê um brinquedo de montar pronto, você entende que alguém pensou e montou as peças. A doutrina espírita usa uma comparação parecida para dizer que, ao ver a natureza e o universo funcionando com regras (dia e noite, plantas crescendo, estrelas no céu), podemos entender que existe uma origem inteligente por trás.
