# Kardec para Crianças

Bem-vindo à documentação do projeto **Kardec para Crianças**.

![Allan Kardec](assets/images/kardec.jpg)

Aqui você encontrará todas as questões do **Livro dos Espíritos** de Allan Kardec, com respostas adaptadas para crianças e com ilustrações.

## Navegação

Utilize o menu lateral para navegar entre as questões.

## Sobre

Este projeto visa facilitar o entendimento do Espiritismo para crianças através de uma linguagem acessível e ilustrações.
